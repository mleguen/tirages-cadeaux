# Tkdo

Tirages au sort de cadeaux.

## Démarrage (développement)

> Prérequis :
> - docker
> - docker-compose
> - openssl
> - node et npm

```
$ make start
```

Après la construction des images et le démarrage des conteneurs Docker, votre navigateur s'ouvrira sur [la page d'accueil](https://localhost).
